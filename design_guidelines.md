# AI Integration Service Provider Website Design Guidelines

## Design Approach
**Reference-Based Approach**: Drawing inspiration from modern technology service providers like Stripe, Notion, and Linear. This approach emphasizes trust, professionalism, and technical sophistication while maintaining approachability for diverse client types.

## Core Design Elements

### Color Palette
**Primary Colors:**
- Deep Navy: 220 45% 15% (main brand color, headers, CTAs)
- Bright Blue: 210 85% 55% (accent for highlights, links)
- Soft Gray: 210 10% 95% (light backgrounds, subtle sections)

**Supporting Colors:**
- Pure White: 0 0% 100% (clean backgrounds, text on dark)
- Charcoal: 220 15% 25% (body text, secondary elements)
- Success Green: 145 65% 45% (case study highlights, success indicators)

### Typography
- **Primary Font**: Inter (Google Fonts) - clean, modern, highly readable
- **Headings**: 600-700 weight, generous line height (1.2-1.3)
- **Body Text**: 400-500 weight, comfortable reading size (16-18px base)
- **Technical Content**: 500 weight for emphasis on capabilities

### Layout System
**Tailwind Spacing Primitives**: 4, 8, 12, 16, 24
- Consistent rhythm using these units for margins, padding, and grid gaps
- Generous whitespace to convey professionalism and reduce cognitive load

### Component Library
- **Navigation**: Clean horizontal nav with subtle hover states
- **Hero Section**: Large, impactful with clear value proposition
- **Service Cards**: Clean grid layout with icons and concise descriptions
- **Case Study Cards**: Image + brief description + results metrics
- **Contact Forms**: Professional styling with clear field labels
- **CTAs**: Bold primary buttons, subtle secondary buttons

## Visual Treatment

### Professional Technology Aesthetic
- **Clean Lines**: Sharp, precise edges and consistent border radii (4-8px)
- **Subtle Depth**: Gentle shadows and elevation for card components
- **Technical Precision**: Grid-based layouts with mathematical spacing
- **Trust Indicators**: Professional imagery, client logos, certification badges

### Content Strategy
**5 Key Sections Maximum:**
1. **Hero**: Compelling value proposition with clear CTA
2. **Services Overview**: Grid of core automation capabilities
3. **Case Studies**: 3-4 success stories with metrics
4. **Trust/Expertise**: Client testimonials, certifications, team credentials
5. **Contact**: Simple form with consultation CTA

### Background Treatments
- **Primary**: Clean white backgrounds with subtle gray section dividers
- **Accent Sections**: Light gray (210 10% 98%) for service highlights
- **No Gradients**: Maintain corporate professionalism with solid colors

## Images
**Hero Image**: Yes - large, professional image showing AI/automation concepts (abstract tech visualization, clean office setting, or data visualization mockup)
**Service Icons**: Simple, consistent line-style icons for each automation type
**Case Study Images**: Clean screenshots or abstract representations of workflow improvements
**Team Photos**: Professional headshots if including team section

## Key Design Principles
1. **Professional Trust**: Every element should reinforce expertise and reliability
2. **Technical Sophistication**: Visual design that appeals to technical decision-makers
3. **Clear Value Communication**: Immediate understanding of AI integration benefits
4. **Conversion-Focused**: Strategic placement of contact forms and consultation CTAs
5. **Scalable Foundation**: Design system that can grow with additional services
